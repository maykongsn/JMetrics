# :mag: JMetrics

> Metrics for Java code

---

This project aims to create a simple static code analyzer that produces some metrics from Java code. 

## JavaParser

This project uses JavaParser to generate the Abstract Syntax Tree (AST) of Java code.

- [JavaParser Repository](https://github.com/javaparser/javaparser/)
